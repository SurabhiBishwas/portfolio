# Public Assets

Place your profile photo here.

## Profile Photo

- **Filename**: `profile-photo.jpg` (or update the filename in `src/components/Hero.jsx`)
- **Recommended size**: 512x512px or larger
- **Format**: JPG, PNG, or WebP
- **Aspect ratio**: Square (1:1) works best for the circular frame

The image will automatically display in the Hero section with a beautiful circular frame and gradient background effect.
